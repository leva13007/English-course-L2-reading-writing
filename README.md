## Structure

```
homework.md                — central homework tracker (all teachers)
vocab.md                   — unified vocabulary for Anki
1_tue_gozde/               — Tuesday classes (Gozde)
  └── YYYY-MM-DD/          — one folder per class date
      ├── class.md          — class notes, topic, materials
      ├── hw.md             — homework assignment
      ├── hw_solution.md    — my solution
      ├── notes.md          — optional ideas/observations
      └── attachments/      — screenshots, scans, handouts
2_wed_jonathan/            — Wednesday online classes (Jonathan)
3_thu_andrew/              — Thursday classes (Andrew)
essays/                    — Writing assignments
resources/                 — Shared scans, handouts, Teams files
_templates/                — Reusable templates (class, hw, hw_solution, notes)
_archive/                  — Old files (pre-reorganization)
```

## Workflow

1. Before class: copy `_templates/class.md` → `N_day_teacher/YYYY-MM-DD/class.md`
2. During/after class: fill in topic, notes, materials
3. Drop screenshots/scans into `YYYY-MM-DD/attachments/`
4. If homework assigned: copy `_templates/hw.md` → same date folder + add row to [homework.md](homework.md)
5. When doing homework: copy `_templates/hw_solution.md` → same date folder
6. Add new words → [vocab.md](vocab.md) under the teacher's section
7. Mark ✅ in vocab.md when word is added to Anki
8. Update status in [homework.md](homework.md): 🔴→🟡→🟢
