## Class Notes

Потрібно написати репор по графікуб який показує зміни якихось показників на протязі якогось періоду часу. Потрібно описати, що відбувається на графіку, які є тенденції, які є висновки. Можна використовувати різні типи графіків, такі як лінійні, стовпчикові, кругові тощо, залежно від того, які дані ви хочете показати. Важливо також враховувати контекст даних і пояснювати, що вони означають для вашої аудиторії.

Report structure:

1. Introduction: Briefly introduce the topic and the purpose of the report.
2. Overview of the main trends: Describe the main trends shown in the graph, such as increases, decreases, or fluctuations in the data.
3. Detailed comparison: Compare different data points or time periods to highlight significant changes or patterns.
4. Reasons and future predictions: Discuss possible reasons for the observed trends and make predictions about future developments based on the data.

Vocabulary, line graph languege:
increase, fall, remain stable, decrease, go down, fluctuate, rise, drop, go up, stay the same, decline, double, grow, halve, improve, jump, plummet, soar, surge, peak, reach a high/low point, level off, show a steady increase/decrease, experience a significant change, etc.

Create past form those verbs to describe past trends:
increased, fell, remained stable, decreased, went down, fluctuated, rose, dropped, went up, stayed the same, declined, doubled, grew, halved, improved, jumped, plummeted, soared, surged, peaked, reached a high/low point, leveled off, showed a steady increase/decrease, experienced a significant change, etc.

Write nouns from those verbs to describe trends:
increase, fall, stability, decrease, fluctuation, rise, drop, growth, improvement, jump, plummet, soar, surge, peak, leveling off, significant change, etc.

Adverbs to describe trends:
significantly, sharply, steadily, gradually, slightly, dramatically, rapidly, moderately, considerably, marginally, etc
